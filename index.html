APPLICATIONS DEVT & EMERGING TECHNOLOGIES

This course provides opportunity for students for students to learn new approach in the creation of web pages using server side scripting language. It provides introduction to dynamic and interactive tools and methods in web development. It also focuses on the use of PHP and MySQLi in the creation of interactive and dynamic webpages with database applications.

Course Objectives

  • To develop a foundational understanding of structured programming concepts and syntax using PHP, enabling students to write basic scripts that follow programming standards.
  • To equip students with practical skills in web development, focusing on integrating PHP into web applications, with an emphasis on debugging and troubleshooting techniques.
  • To guide students in designing program flowcharts and structuring code for solving simple computing problems, ensuring the solutions meet the specified functional requirements of a website.

Course Outcomes

  • 1. Explain the fundamental principle structured programming language using PHP Script.
  • 2. Apply the fundamental principles of web development in creating computing application with debugging techniques.
  • 3. Design program flow and program structure for simple computing problems that meets specific requirements for the program website.
